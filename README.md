# Humor Scan 🔍

Projeto de desenvolvimento web desenvolvido como parte das atividades acadêmicas da graduação em *Sistemas para Internet na Uncisal*.

### 📝 Sobre o Projeto:
O *Humor Scan* é um quiz interativo que utiliza lógica de programação para realizar uma "detecção" de humor baseada nas respostas do usuário. O objetivo principal foi aplicar conceitos fundamentais de front-end e interatividade.

### 🛠 Tecnologias Utilizadas:
* *HTML5:* Estruturação semântica da interface.
* *CSS3:* Estilização responsiva e design de experiência do usuário (UX).
* *JavaScript:* Manipulação do DOM e lógica dos resultados do quiz.

### 🚀 O que este projeto demonstra:
* Capacidade de criar interfaces interativas.
* Organização de código e separação de responsabilidades (HTML/CSS/JS).
* Resolução de problemas através de lógica de programação.
